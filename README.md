koa-gulp-crud
=============

A simple [Koa](http://koajs.com/) example app.

- RESTful endpoint with CRUD functionality
- Endpoint-level tests via `mocha` and `co-supertest`
- gulpfile running and refreshing server and tests on file-save via `gulp`
